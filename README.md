## Viral Video Production

The transcript describes short-form video creation using existing viral videos as inspiration.

### Workflow described

1. Find a viral TikTok or Instagram format.
2. Download or obtain the video.
3. Extract the first frame with LastFrame.ai.
4. Generate a replacement first frame using the avatar.
5. Use an AI video/motion tool to animate the avatar based on the source movement.
6. Edit the output in CapCut.
7. Add a hook text overlay.
8. Export in 4K / 60 FPS.

### Copyright note

Do not copy or re-upload someone else’s creative work without rights. Use original footage, licensed footage, or recreate formats without copying protected expression.

## Account Setup and Warm-Up

The transcript recommends warming up social accounts before posting.

### Hardware/account setup mentioned

- Use a phone above iPhone SE 2020.
- Use one account per device.
- Use a 4G network.
- Use a US SIM if targeting a US audience, according to the transcript.

### Seven-day warm-up sequence described

| Day | Action |
|---:|---|
| 1 | Create the account. Do not add profile changes. Scroll 30–60 minutes. Like some content. Do not comment. |
| 2 | Add bio and profile picture. Do not add monetization links yet. |
| 3–4 | Scroll, post lightly, follow 2–3 people per day. |
| 5 | Start posting stories, roughly one every other day. |
| 7+ | Account is considered warmed up by the speaker. |
| 10–14 | Begin posting Reels more consistently. |
| 14+ | Add link in story, according to the transcript. |

### Posting guidance from transcript

- Prefer one high-quality video per day over multiple weak videos.
- Avoid posting low-quality videos that may reduce reach.
- Use clear hooks and “shock value” style content.
